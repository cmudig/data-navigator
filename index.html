Data Navigator Static PNG Example

Welcome to our first major case study and prototype for Data Navigator. This case study demonstrates that a static PNG can be navigated along two different tree-like structures at once. In addition, we are demonstrating that commands for movement may be typed into a text input, spoken, gestured using one hand or you may also use keyboard keys, touch swipes, mouse drags, and screen readers to navigate the content.

For now, we are not providing instructions for how to operate every control in this case study, but in a later release we will explain how to use each of these different input modalities.
